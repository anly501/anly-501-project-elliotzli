Data Gathering

SAMHSA

Link

To understand my topic better, I used SAMHSA, Substance Abuse and Mental Health Services Administration, as a foundation. I collected 22 sets of data from SAMHSA. Content includes but is not limited to: demographic, geographic, and socioeconomic statistics on populations with mental health issues, attempted suicide, populations with mental health treatments, substance use, etc.

Raw data is stored under the file name pre-clean data. Here are some screenshots of the datasets I collected.

Detailed information about my collection of SAMHSA can be found here: Data Table of Content

All raw data can be found here: pre-clean data

Twitter API

I used Twitter API in R to gather data about public opinions on relevant topics. I used keywords:'depressive disorder','high school drug', and 'young suicide' to gather tweets and formed three wordclouds. Visualizations are shown in the tab Exploring Data.

twitter search r depressive disorder

twitter search r high school drug

twitter search r young suicide

I used Twitter API in Python to gather tweets on topics 'mental health teenager','substance use young people', and 'lgbtq youth people', and analyzed sentiments.

twitter api youth:mental:substance:lgbtq.ipynb
