Details

Numbers in Thousands

1. I merged two tables on demopraghic and geographic & socioeconomic characteristics. Both tables contain data in the time period of 2019-2020, and measures the population that uses illicit drugs in the time period.

2. I dropped columns: 'Aged 12+','Aged 12+.1','Aged 18+','Aged 18+.1' because these columns are the sum of data of the rest of the dataset. It will skew my data when I start running models.

3. I dropped rows that either further in detail in the table. For example, contained no mumeric information or specifically illustarted further in the table. For example, I dropped the data entry of obtaining the sum data about alcohol, and kept subsections.

4. I replaced row names in order to make the data table more aesthetically pleasing. For example, 'Full-Time' to 'Full-Time Employment'.

5. I standardized all numeric data type fromobject to floating 64.
