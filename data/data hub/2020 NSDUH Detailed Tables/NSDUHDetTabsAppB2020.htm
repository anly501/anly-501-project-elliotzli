Appendix B: List of Tables

The table type indicators appear in parentheses after each table number. Refer to the “Table Presentation” section of the Introduction for more details about the organization and numbering of the tables, including some exceptions to the primary table types. The primary table types are as follows:

  • A = Numbers in Thousands
  • B = Percentages
  • C = Standard Errors for “A” Tables
  • D = Standard Errors for “B” Tables
  • N = Survey Sample Size

The coronavirus disease 2019 (COVID-19) pandemic made 2020 a unique year within the history of the National Survey on Drug Use and Health (NSDUH). The 2020 estimates for most measures were calculated using data from Quarters 1 and 4 of 2020 and are labeled as “2020” in table titles for brevity. For a few tables, estimates for measures were calculated using data from only Quarter 4 of 2020. These estimates are labeled as “Quarter 4, 2020” in table titles. Refer to the “Considerations for the 2020 NSDUH Detailed Tables” section of the Introduction for more details.
